# Driver profile schemas

Артефакты схемы и типов для экрана профиля водителя.

## Содержимое
- `driver_profile_screen.schema.json` — JSON-схема текущего экрана профиля.
- `driver_profile.types.ts` — TypeScript-типы для payload/состояний.
- `readiness_summary.schema.json` — контракт summary-блока readiness и `blocking_reasons[]` для Driver Profile v2.
