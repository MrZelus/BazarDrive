## Как использовать

- `docs/driver_master_ux_map.md` использовать как верхнеуровневую карту.
- `docs/driver_order_flow.md` использовать как source of truth по сценарию заказа глазами водителя.
- `docs/driver_shift_flow.md` использовать как source of truth по жизненному циклу смены.
- `docs/driver_menu_map.md` использовать как карту навигации Telegram / Web.

## Рекомендация

После согласования схем стоит заменить временные ссылки ChatGPT/FigJam на постоянные ссылки рабочего пространства Figma, если они будут перенесены в основной файл проекта.
