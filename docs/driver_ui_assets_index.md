# Driver UI Assets Index

Этот документ собирает в одном месте все актуальные FigJam-артефакты по driver UX для BazarDrive.

Документ нужен как единая навигационная точка между:
- сценариями;
- flow-схемами;
- картами экранов;
- wireframes;
- mobile low-fi;
- текстами интерфейса и CTA.

## 6. Как использовать в проекте

### Product / UX
Использовать как карту согласования пользовательских сценариев и структуры экранов.

### Frontend
Использовать для сборки экранов, состояний, CTA и навигации.

### Telegram bot
Использовать для синхронизации экранной логики с короткими бот-сценариями и статусами.

### Backend / API
Использовать для согласования переходов состояний, блокеров допуска и статусов заказа / смены.

### QA
Использовать как основу для сценариев ручного и автоматизированного тестирования.
