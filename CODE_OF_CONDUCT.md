# Code of Conduct

## Our Commitment

We aim to create an open, respectful, and professional environment for all contributors.

## Expected Behavior

- Be respectful and constructive
- Focus on the problem, not the person
- Provide actionable feedback
- Keep discussions relevant to the project

## Unacceptable Behavior

- Harassment, insults, or discrimination
- Spam or irrelevant content
- Disruptive behavior in discussions or PRs

## Enforcement

Project maintainers may take appropriate action in response to violations.

## Scope

This Code of Conduct applies to all project spaces, including issues, pull requests, discussions, and external communication related to BazarDrive.
